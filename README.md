# Ciclo Hamiltoniano

## Funcionamento

```py
    python main.py
```

Irá printar False, se não houve um caminho, contudo se for True, printará também o caminho

Em ambos, os casos é possível ver o grafo através do arquivo `graph.html` através do Browser
